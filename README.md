## 🔍 What pixelproof looks for

pixelproof scans images for signs like these:

- Inconsistent lighting or shadows  
- Sudden changes in pixels that don't match the rest of the image  
- Abnormal or missing EXIF data (details about when and how the photo was taken)  
- Patterns common in AI-generated images  
- Hidden messages or data embedded in the picture files
